## рҹ”Ҙ Feature Highlights

</div>

<br/>

**рҹ“Ҡ Rich Data Visualizations**
> Balance trend area charts В· Weekly cash flow bars В· Net worth projection В· Radial spending breakdown вҖ” all powered by Recharts, all consuming live context state.

**рҹ§ҫ Transaction Engine**
> Add, edit, delete, bulk-delete, export to CSV. Search by merchant, category or note. Filter by type and status. Sort by date, amount or merchant. Paginated. Animated.

**рҹ”җ Role-Based Access Control**
> Three roles, one context. Permissions cascade from `RoleContext` to every button, checkbox, and action in the UI вҖ” no scattered conditionals, no hacks.

| | рҹ‘‘ Admin | вңҸпёҸ Editor | рҹ‘ҒпёҸ Viewer |
|:--|:--:|:--:|:--:|
| View data | вң… | вң… | вң… |
| Add transactions | вң… | вң… | вқҢ |
| Edit transactions | вң… | вң… | вқҢ |
| Delete transactions | вң… | вқҢ | вқҢ |
| Export CSV | вң… | вң… | вқҢ |

**рҹӨ– AI Financial Assistant**
> Chat interface backed by `claude-sonnet`. Sends your financial context вҖ” balances, spending patterns, goals вҖ” so Claude gives answers that are actually relevant to *your* money.

**рҹ’ҫ Cache-First State Strategy**
> `FinanceContext` hydrates from `localStorage` instantly on mount, checks staleness (> 5 min), then background-refreshes. Mirrors real SWR behavior without a library.

**1 вҖ” Clone**
```bash
git clone https://github.com/adrijaDatta/Finance-tracker-.git
cd finance-dashboard
```

**2 вҖ” Install**
```bash
npm install
```

**3 вҖ” Add your API key**

